
Schritt 1: Raspbian runterladen und auf Raspberry packen
 - http://jankarres.de/2012/08/raspberry-pi-raspbian-installieren/
 - an Bildschirm anschließen
 - "Expand Filesystem"
 - "Internationalisation Options" -> Locale / Keyboard Layout auf DE machen, wenn gewünscht
 - "Advanced Options" -> SSH -> enable

Schritt 2: WLAN aktivieren
 - Datei /etc/network/interfaces auf SD-Karte packen

SSH Key generieren
 $ ssh-keygen -t rsa
 $ cat ~/.ssh/id_rsb.pub
 - Auf Gitlab eintragen

Schritt 3: Boatcontrol installieren:
 $ sudo apt-get install -y git
 $ sudo mkdir /www/ ; sudo chmod 777 /www
 $ cd /www
 $ git clone git@mdoetsch.de:root/boatcontrol.git
 $ cd boatcontrol/scripts
 $ sudo sh install.sh

Schritt 7: Autostart script
 - Datei "/home/pi/.bashrc" öffnen
 - Folgende Zeile eintragen am Ende (ohne "): ". /www/boatcontrol/scripts/alias.sh"

Schritt 8:
 - Datei "/etc/cron.d/boatcontrol" anlegen/öffnen
 - Folgende Zeile eintragen (ohne "): "@reboot cd /www/boatcontrol/scripts/ ; sudo sh ./start.sh"
 - Wichtig: mindestens eine leere Zeile am Ende der Datei muss vorhanden sein
